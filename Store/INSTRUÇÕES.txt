*CONFIGURANDO O BACKUP DO Store

1. Copiar o arquivo "BackupStore.bat" para a pasta do Store ("C:\Program Files\Store" ou "C:\Program Files (x86)\Store")

2. Clicar com o botão direito do mouse no arquivo "BackupStore.bat" e selecionar a opção Editar

3. Substituir "PREENCHER NOME DO BANCO DE DADOS" pelo nome do banco de dados (Verificar o nome do banco de dados utilizando o ListarBancos.bat)
Obs: o nome do banco de dados deve obedecer os caracteres minusculos e maiusculos, caso contrario não será possivel identificar o banco

4. Clicar com o botão direito do mouse no arquivo "AgendarTarefa.bat" e selecionar a opção Editar

5. Substituir "PREENCHER COM O HORARIO DO BACKUP" com o horario que o backup deverá ser afetuado (O horario deve ser preenchido no formato de 24h, ex: 8:00, 16:00, 14:30)

6. Salvar e executar o arquivo "AgendarTarefa.bat"
------------------------------------------------------------------
*CONFIGURANDO O BACKUP DO Store EM REDE

1. No micro que será destino do backup, executar "CompartilhaPastaBackup.bat" (Clicar com o botão direito do mouse e selecionar a opção "Executar como administrador")

2. No servidor, copiar o arquivo "BackupStoreRede.bat" para o pasta do Store ("C:\Program Files\Store" ou "C:\Program Files (x86)\Store")

3. Alterar o nome do arquivo "BackupStoreRede.bat" para "BackupStore.bat"

4. Clicar com o botão direito do mouse no arquivo "BackupStore.bat" e selecionar a opção Editar

5. Substituir "PREENCHER NOME DO BANCO DE DADOS" pelo nome do banco de dados (Verificar o nome do banco de dados utilizando o ListarBancos.bat)
Obs: o nome do banco de dados deve obedecer os caracteres minusculos e maiusculos, caso contrario não será possivel identificar o banco

6. Subtituir todos os "NOME DO MICRO DE DESTINO DO BACKUP" pelo nome do micro que será o destino do backup. (Para verificar o nome, executar o arquivo "VerificaNome.bat" no micro que será o destino do backup)

7. Clicar com o botão direito do mouse no arquivo "AgendarTarefa.bat" e selecionar a opção Editar

8. Substituir "PREENCHER COM O HORARIO DO BACKUP" com o horario que o backup deverá ser afetuado (O horario deve ser preenchido no formato de 24h, ex: 8:00, 16:00, 14:30)

9. Salvar e executar o arquivo "AgendarTarefa.bat"
